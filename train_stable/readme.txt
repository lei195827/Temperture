1.1-27.xls为设备稳定运行时采集的与温度相关的特征；
其中CPU0-3表征设备上cpu的温度值， Brightness表示设备屏幕亮度（建议可暂时不考虑这维特征）;
module Humidity表示内部温度传感器读取到的湿度， module temperature为温度值， lm75为设备上外接板载温度值
GT_Temp为真实环境实际温度

2.1-27包含了常温环境下及温箱环境下采集数据，常温环境相对稳定，基本为室温温度，波动不大，温箱数据则包含了0-40度的温度数据，采集策略为每1-2小时升温5度，需要注意的是温箱升温较快，而设备端温度升温较为缓慢，这点是有滞后性

3.28.xls为室温环境下，设备端通话过程中，温度变化数据，在设备使用过程中，设备容易发烫导致内部温度传感器升温,其中 IsTalking=1表示正在通话过程， IsTalking=0结束通话
  29.xls与28对应，为同一时刻下，未开启通话过程中另外一台设备的温度值
